# Harukaze no Kaeri

*A return carried on the spring wind.*

## About

Harukaze no Kaeri is a bright top-down pixel adventure game set in a peaceful rural village.  
Wield a blade, awaken gentle magic, and journey through sunlit fields to restore someone lost.

## Features

- Player-centered camera zoom and follow system
- Smooth movement mechanics
- Light sword combat
- Bright pixel-art aesthetic

## Built With

- HTML
- CSS
- JavaScript
